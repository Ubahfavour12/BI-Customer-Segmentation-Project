# Customer Segmentation Using Machine Learning

## Business Intelligence and Data Mining Project

This project applies clustering techniques to segment mall customers based on income and spending behaviour.

## Objectives

- Identify customer segments
- Analyse purchasing behaviour
- Provide marketing recommendations

## Dataset

Mall Customer Segmentation Dataset (Kaggle)

## Techniques Used

- Exploratory Data Analysis
- K-Means Clustering
- Data Visualisation
